## Selected Software

## Project Description
This project is an audit of an open-source software (VLC).  
It includes analysis of its origin, license, ethics, Linux working, ecosystem, and comparison with proprietary software.

## Scripts Included

### 1. script1.sh – System Identity
Displays system information like kernel, user, uptime, and date.

### 2. script2.sh – Package Checker
Checks whether VLC is installed on the system or not.

### 3. script3.sh – Directory & Permissions
Shows directory size and file permissions.

### 4. script4.sh – Log Analyzer
Counts number of "error" entries in system log files.

### 5. script5.sh – Manifesto Generator
Takes user input and generates a small open-source manifesto.

## How to Run Scripts

1. Give permission:
   chmod +x script.sh

2. Run script:
   ./script.sh
